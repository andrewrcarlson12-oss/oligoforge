# Molecular Vulnerability Record guide

An OligoForge Molecular Vulnerability Record links a registered assay, detecting DriftGuard scan, baseline/current snapshots, affected records/components, modeled observation, predicted consequence, evidence strength, completeness, model versions, limitations and review/disposition history.

Identifiers use `OFVR-YYYY-LOCALIDENTIFIER`. They are deterministic for the declared issuance year and reason fingerprint. OFVR is an OligoForge-local record and must not be presented as a recognized external vulnerability standard.
