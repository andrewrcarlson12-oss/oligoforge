Conservation & discrimination

Per-base conservation of each oligo across a target set, and how cleanly it mismatches an off-target set (e.g. one genus vs a closely related genus). Paste FASTA or pull from NCBI.

In-silico PCR

BLASTs both primers and reports predicted products: same subject, opposite strands, convergent, in size range. The intended target should be the only product at the expected size. Remote BLAST is queued at NCBI (slow); local needs BLAST+ and a genome DB.

Standard curve & efficiency

Paste standard-curve results, one "quantity, Cq" per line (copies or ng). Repeat a quantity for replicates; leave Cq blank or NA for a non-detect. Reports efficiency, R², dynamic range, and a practical LOD (MIQE).

Target → assay

Type an NCBI target (organism AND gene), it fetches sequences, designs a primer/probe set on the most complete one, and scores conservation across the whole set plus discrimination against an off-target set. A screen, not a final answer — confirm winners in OligoAnalyzer. AT-rich parasite mtDNA needs a low-Tm parasite profile. Timing: without BLAST this is one NCBI fetch plus design — usually well under a minute; if it ever stalls it now fails at 45 s rather than hanging. Ticking BLAST adds a remote NCBI search that can take several minutes. Fewer sequences to fetch is faster.

Reference-gene stability

geNorm M + pairwise V (Vandesompele 2002) and BestKeeper Cq SD/CV. Paste your SYBR-screen Cq table: one gene per line — name then one Cq per sample, space or comma separated. Needs ≥2 genes and ≥3 samples. MIQE wants ≥2 validated reference genes.
